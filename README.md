# Apricity

Apricity is novel-writing organization software for fiction writers.

It helps authors manage the parts around the manuscript: characters, locations, scenes, relationships, timelines, world details, and continuity facts.

Apricity is not a text editor.

## Requirements
- C++20
- wxWidgets 3.3
- Premake5
- Visual Studio 2026

## Building

Generate the Visual Studio project:
```
premake5 vs2026
```
Open the generated solution in Visual Studio and build the project.
