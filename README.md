# Apricity

Apricity is a desktop story control panel for planning novels and other sorts of stories. It is not a manuscript editor; it organizes the supporting structure around a writing project.

## Building

Apricity uses C++20, wxWidgets, SQLite, and Premake5.

The supported Windows development path is Visual Studio 2026:

```bat
premake5 vs2026
```

Solution will be at:

```text
Build/ProjectFiles/Apricity.slnx
```
